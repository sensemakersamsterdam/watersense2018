# RTOSSewer

Sensing Water quality in a variety of ways. Used by Watersense2018 team.


## Tested on hardware:

- SODAQ ONE-EU-RN2483-V3 https://shop.sodaq.com/sodaq-one-eu-rn2483-v3.html
- BMP280 sensor https://www.adafruit.com/product/2651
